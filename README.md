# Retail Sales & Customer Analytics Dashboard

## Overview
End-to-end data analytics project analyzing 1,000 
retail transactions across FY 2023.

## Tools Used
- Excel — Data cleaning & preparation
- SQL (SQLiteOnline) — Analysis & RFM segmentation  
- Power BI — Interactive dashboard & visualization

## Key Insights
- Total Revenue: $456,000 across FY 2023
- Peak Month: May at $53,000
- Top Category: Electronics at $157K (34.41%)
- Avg Order Value: $456 per transaction
- August lowest sales ($24K) — seasonality pattern
